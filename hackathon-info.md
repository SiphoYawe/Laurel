# Commit To Change: An AI Agents Hackathon

> Build AI that turns New Year's resolutions into real results.

## Overview

| Detail | Info |
|--------|------|
| **Start Date** | January 13, 2026 |
| **Duration** | 4 weeks |
| **Format** | Online |
| **Total Prize Pool** | $30,000 USD |
| **Organizer** | Encode Club |
| **Main Sponsor** | Comet (Opik) |

## What You'll Build

AI agents and LLM-powered apps that help people stick to their goals. Whether it's crushing productivity, building healthier habits, or making better financial decisions, your code could be the difference between another broken resolution and lasting change.

## What You Get

- **Expert Workshops**: Learn observability, evaluation, and agent optimization from Comet's team
- **Premium Tools**: Credits and support from Google, Vercel, and other partners
- **Direct Mentorship**: Technical support throughout the hackathon via Discord
- **Real Impact**: Build tools people will actually use to achieve their goals
- **$30,000 in Prizes**: Compete across six categories

## Who Should Join

Developers with:
- Basic understanding of LLMs and AI agents
- Experience building software applications
- Python or TypeScript knowledge

Beginner-friendly if you've worked with LLMs before. If you're new to AI agents, expect an intermediate challenge with strong learning support.

## Prize Categories

### 1. Best Use of Opik - $5,000
**Supported by Comet**

Showcase exceptional implementation of evaluation and observability in your AI system. Demonstrate how you use Opik to systematically track experiments, measure agent performance, and improve system quality with data-driven insights.

**Example Projects:**
- Chat agent with online LLM-as-judge evaluations
- Automated prompt/agent tuning loop using Opik Agent Optimizer
- Guardrailed compliance summarizer with PII/red-flag detection
- Regression test suite for feature-flagged LLM app
- RAG QA bot with Opik tracing + evals
- Multi-tool coding agent with tool-selection accuracy tracking

**Note:** This award can be won alongside a Best in Category prize. Maximum prize per team is $10,000.

## Judging Criteria (All Categories)

1. **Functionality**: Does the app actually work as intended? Are the core features implemented, stable, and responsive?
2. **Real-world relevance**: How practical and applicable is this solution to real users' lives and real-world New Year's goals?
3. **Use of LLMs/Agents**: How effectively does the project use LLMs or agentic systems (reasoning chains, autonomy, retrieval, tool use)?
4. **Evaluation and observability**: Has the team implemented ways to evaluate or monitor their system's behavior (metrics, human-in-the-loop validation, Opik integration)?
5. **Goal Alignment**: Category-specific alignment with the prize theme

### Additional Criteria for Best Use of Opik:
- How well is Opik integrated into the development workflow?
- Does the team use Opik to produce meaningful insights?
- Are Opik dashboards, metrics, or visualizations clearly presented for judging?
